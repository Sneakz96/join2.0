What is Join?

Join is a dynamic Kanban board. In japanese 'Kanban' means 'Visual Signal' or 'Visual Card'. The main purpose of Kanban is to help and improve manufacturing efficiency and create a transparent work process for everyone involved.

How to use it

1.

In the Add-Task-section you can create a new task. It automatically will displayed on the Board in the To-Do Section.

2.

You can change the status of a task by drag and drop. Grab the task to change and drop it in the collum of the new status. Done Tasks will automatically delete after 3 days.

3.

For more details of a task or to edit it, click on the task. In the opened Overlay you can see the whole information about a task. You can also edit it, by clicking the edit button.

4.

In the Contact-section you can add new contacts, which where displayed in the Contact-List. For more details click on them. When a contact get opened you can edit it.
